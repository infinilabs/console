## 简介

## 文档详情

点击单行文档左侧的 " > " 按钮可展开查看文档的详细信息，并支持 Table 和 JSON 两中方式查看。

{{% load-img "/img/screenshot/v1.29/datainsight/discover-table.png" "" %}}

同时可对文档数据进行编辑、删除等操作。
